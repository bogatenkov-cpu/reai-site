# RE:AI — сайт AI-консалтинга для недвижимости

## О проекте

Лендинг для продуктового бренда RE:AI — AI-внедрения для компаний недвижимости.
Кофаундеры: Евгений Богатенков и Кирилл Карманов.

## Структура

```
reai-site/
├── src/
│   └── index.html        ← основная страница (HTML + inline CSS)
├── public/
│   └── images/           ← сюда положить скриншоты и фото
├── CLAUDE.md             ← инструкции для Claude Code
└── README.md             ← этот файл
```

## Стек

Сейчас — чистый HTML/CSS, один файл. Для деплоя можно:
- **Vercel** — статика, `vercel deploy` из корня
- **GitHub Pages** — push в repo, включить Pages
- **Cloudflare Pages** — аналогично

Если нужен фреймворк (для блога с кейсами, маршрутизации, CMS) — можно перевести на:
- **Astro** — идеально для контентных сайтов
- **Next.js** — если нужна динамика
- **Hugo** — если нужна максимальная простота
