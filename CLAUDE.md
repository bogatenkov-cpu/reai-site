# CLAUDE.md — инструкции для Claude Code

## Контекст проекта

Это лендинг для RE:AI — бренда AI-консалтинга в недвижимости.
Кофаундеры: Евгений Богатенков (AI & Marketing) и Кирилл Карманов (Product & Technology).

Целевая аудитория: руководители и CMO компаний недвижимости (девелоперы, агентства, брокеры), преимущественно русскоязычный рынок + международные компании.

## Структура файлов

- `src/index.html` — главная страница, весь HTML + CSS inline
- `public/images/` — папка для изображений
- Деплой планируется на Vercel или аналог

## Дизайн-система

Тёмная тема с многослойными серыми фонами:
- `#0c0c0c` — глубокий (hero, CTA)
- `#121212` — базовый (кейсы, команда)
- `#181818` — приподнятый (карточки задач, продукты, процесс)
- `#252525` — элевированный (манифест)
- `#1e1e1e` — карточки

Акцент: `#c8ff00` (салатовый/лайм)
Текст: `#eeebe4` (основной), `#ccc8bf` (вторичный), `#9e9a92` (третичный)

Шрифты:
- Display: Instrument Serif
- Mono: Space Mono
- Body: DM Sans

## Тон копирайта

Прямой, без маркетингового bullshit. Без JTBD-жаргона на поверхности (но структура JTBD под капотом). Говорим о проблемах клиента и результатах, не о технологиях. Уверенный тон людей, которые делают это каждый день у себя.

## Правила при доработке

1. Не добавлять белый фон — всё в тёмных серых
2. Текст должен быть читаемым — минимум `#9e9a92` для подписей
3. Сохранять салатовый `#c8ff00` как единственный акцент
4. Мобильная адаптация обязательна
5. Не использовать JS-фреймворки без явного запроса — пока чистый HTML/CSS
6. Русский язык по умолчанию
